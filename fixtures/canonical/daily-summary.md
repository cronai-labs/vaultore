# Daily Summary Generator

Runs daily to create a summary of recent vault activity.

## Step 1: Gather Recent Notes

```ore:ts id=gatherNotes
// In a real implementation, this would use vault.list() 
// to get recently modified notes
const today = new Date().toISOString().split('T')[0];

// Simulated: read a few notes
const notes = [];

// Try to read daily note
try {
  const daily = await vault.read(`Daily/${today}.md`);
  notes.push({ type: 'daily', content: daily });
} catch (e) {
  console.log('No daily note found');
}

// Return what we found
JSON.stringify({
  date: today,
  noteCount: notes.length,
  notes: notes
});
```

## Step 2: Generate Summary

```ore:ai id=summarize depends=[gatherNotes]
Based on today's notes, create a brief end-of-day summary.

Notes data:
{{gatherNotes}}

Include:
1. Key accomplishments
2. Open items
3. Notable thoughts or ideas

Keep it concise (3-5 bullet points max).
```

## Step 3: Save Summary

```ore:ts id=saveSummary depends=[summarize, gatherNotes]
const data = JSON.parse(cell("gatherNotes"));
const summary = cell("summarize");

const filename = `Summaries/${data.date}-summary.md`;

const content = `---
created: ${new Date().toISOString()}
type: daily-summary
generated: true
---

# Daily Summary - ${data.date}

${summary}

---
*Generated by VaultOre at ${new Date().toLocaleTimeString()}*
`;

await vault.mkdirp("Summaries");
await vault.write(filename, content);

`Summary saved to ${filename}`;
```
